#1. Grid System introduction in Bootstrap.

container - the job of the container is to put whatever the content it contains center that content.
well class in the Bootstrap provide good view to the text.As happening here. This enclosing template is due to this well class.
row class is used to create the rows in the Bootstrap grid system.
BreakPoint is the point where the columns get break into the multiple rows. There are following classes which provides different breakpoints.

  • lg in col-lg-(some number) - represents large. Breakpoint will be small.
  • lgmd in col-md-(some number) - represents medium. Breakpoint will be medium.
  • lgsm in col-sm-(some number) - represents small. Breakpoint will be small.
  • lgxs in col-xs-(some number) - represents extra small. There will be no breakpoints.
